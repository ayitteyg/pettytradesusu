Project Overview

Project Name: Petty Trading SUSU App
Description: The App is a full-stack web application designed to provide a digital alternative to manual record-keeping for petty trading personnel or groups who engage in some form of susu among themselves, making it easier for members to access their savings records and other transactional data.

Tech Stack

Angular + Material UI Django REST Framework PostgreSQL (Neon) Paystack (MoMo payments) Render (Hosting) JWT Authentication

App Features

  • Sign up
  • Login
  • Member dashboard (savings, withdrawals, current balance)
  • Make deposit via MoMo or cash
  • Request withdrawal
  • Request loan (if eligible) and repayment features
  • Update profile
  • Additional features for account Officer
